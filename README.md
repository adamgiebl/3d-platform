## Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

## Setup & Running

### Server

```bash
cd server
npm install
npm run dev
```

The server will run on `http://localhost:3000` by default.

### Client

```bash
cd client
npm install
npm start
```

The client will run on `http://localhost:3001` and automatically open in your default browser.
